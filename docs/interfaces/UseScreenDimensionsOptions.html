Interface UseScreenDimensionsOptions

useScreenDimensions hook provides the dimensions of the screen for React Native development. It's instrumental in monitoring the current screen size, aiding in real-time screen manipulations.

In many scenarios, it's pivotal to have an understanding of the screen's dimensions for layout, design responsiveness, and dynamically rendering or positioning components based on the available space.

Example

const { width, height } = useScreenDimensions();

Param: options

Configuration options for the hook.

Param: options.scaleType

Specifies whether to track the "window" or "screen" dimensions.

Returns

An object containing the width and height of the screen.

    scaleType?: "window" | "screen";
